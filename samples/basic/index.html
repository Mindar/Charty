Basic Charty Examples

This page demonstrates the usage of Charty with some basic examples. Charty can do much more complicated things, but if all you want to do is add a chart to a site this page is all you'll need to read.

Barebones Example

Before we can add charty elements to our html we need to add the Charty.js file to our html page. It is recommended that you do this asynchronously so that it doesn't block page loading. You can do that with this simple one-liner:

<script src="charty.js" type="text/javascript" async></script>

Now that we have Charty in our page, we can start with a really simple example. The first thing we'll do is simply add a point chart to our document and fill it with points from a csv file. This - once again - is a simple one-liner:

<charty-pointchart src="data.csv"></charty-pointchart>

Adding Titles and Descriptions

For our next demo we're going to add a title and short description to our diagram. HTML 5 introduced two new semantic tags: figure and figcaption. You don't have to use these tags, but using them adds additional semantic information to your page, which is generally nice to have. If you wish to not use the figure or figcaption tag, you could for example put the charty tag inside a div container and then add a normal paragraph below it. However the figure and figcaption are a perfect fit for our purpose and therefore we're going to use them in this example.

<figure>
    <h3>Title!</h3>
    <charty-pointchart src="data.csv"></charty-pointchart>
    <figcaption>Figure 1.01: description for this chart</figcaption>
</figure>

As you see, using Charty for static charts is really simple. But Charty can easily handle more complex scenarios such as interactive charts, or charts that display streaming data. Follow the links to find out how to use Charty for that.
